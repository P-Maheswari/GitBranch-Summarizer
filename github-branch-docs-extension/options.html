Branch Docs Settings

Configure your API keys and preferences. These are stored locally in Chrome and never sent anywhere except the respective APIs.

Groq API

Get your key at console.groq.com. Free tier allows ~30 RPM for large models.

Larger models produce better docs but use more RPM quota.

Delay between Groq requests. Increase if you hit rate limits. Default 1400ms ≈ 43 RPM.

GitHub

Without a PAT, GitHub limits you to 60 API requests/hour. A PAT raises this to 5,000/hour. Requires repo scope for private repos, or no scopes for public.

Advanced

Higher values produce more detailed docs but use more Groq quota.
